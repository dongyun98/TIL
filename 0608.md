### GitHub 내 기록 가져오기
1. `git pull origin main`
2. `git add .`
3. `git commit -m ' ' `
4. `git push origin main`


### 보조 브렌치
- `master` : 제품으로 출시될 수 있는 브렌치
- `develop` : 다음 출시 버전을 개발하는 브랜치
- `release` : 이번 출시버전 준비하는 브랜치
